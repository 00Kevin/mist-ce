Run the following commands on your server:

  1. sudo mkdir -p /opt/mistio-collectd
  2. cd /opt/mistio-collectd
  3. sudo wget https://mist.io/core/scripts/deploy_collectd.sh
  4. sudo chmod +x deploy_collectd.sh
  5. sudo ./deploy_collectd.sh {{view.monitor_server}} {{view.uuid}} {{view.passwd}}

Or if you prefer, this one liner

  • sudo mkdir -p /opt/mistio-collectd && cd /opt/mistio-collectd && sudo wget https://mist.io/core/scripts/deploy_collectd.sh && sudo chmod +x deploy_collectd.sh && sudo ./deploy_collectd.sh {{view.monitor_server}} {{view.uuid}} {{view.passwd}}

After you've run the above commands, refresh the page and monitoring graphs will appear!
